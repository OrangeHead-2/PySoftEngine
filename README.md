# PySoftEngine

A full-stack, zero-dependency Python graphics & teaching engine.
- Advanced scene scripting, animation, shaders, physics, GUI, ASCII, and web export.

## Features

- **Shader pipeline:** Vertex/fragment, custom lighting, Python-callable shaders
- **Scene scripting:** Scene graph, nodes, transforms, DSL
- **Animation:** Keyframes, skeletal, threaded playback
- **Physics:** Bounding box, force, gravity, collision, multithreaded
- **GUI:** Tkinter visualizer, live debug panel, step mode
- **ASCII:** Real-time Z-buffered terminal renderer
- **Web:** PyScript/pyodide hooks for browser rendering
- **WASM:** Build pipeline (simulated)
- **Educational:** Step-through, inspect stages, matrices, culling, and more

## Usage

Integrate modules in your app, or run the GUI/ASCII renderers directly.
All code pure Python, no 3rd-party dependencies.
